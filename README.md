# Wrift

基于 Swift Package 写的一个脚本工具，主要辅助自己写博客使用。

## Unsage

安装：

```sh
cd Wrift
make install
```

## Scripts

用来新建 md 文件，插入 Front Matter

```sh
wrift new filename "title name"
```
